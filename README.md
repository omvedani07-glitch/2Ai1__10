## ❓ Problem Statement
-The objective of this project is to predict whether a passenger survived
- or not during the Titanic disaster based on various features such as 
-age, gender, passenger class, fare, etc. This is a binary classification
- problem where the output is either Survived (1) or Not Survived (0). 

---

## 📊 Dataset Description
-The dataset used in this project is the famous Titanic dataset.
-Features include:
-PassengerId
-Pclass (Ticket class)
-Name
-Sex
-Age
-SibSp (Number of siblings/spouses aboard)
-Parch (Number of parents/children aboard)
-Ticket
-Fare
-Cabin
-Embarked
-Target Variable:
-Survived (0 = No, 1 = Yes)
---

## 🛠️ Data Preprocessing Steps
Handled missing values (Age, Embarked, etc.)
Dropped irrelevant columns (Name, Ticket, Cabin)
Converted categorical data into numerical format (Sex, Embarked)
Feature selection
Data scaling using StandardScaler
Train-test split performed
